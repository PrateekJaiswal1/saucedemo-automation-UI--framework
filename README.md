# SauceDemo Automation Framework

![CI](https://github.com/PrateekJaiswal1/saucedemo-automation-UI--framework/actions/workflows/test.yml/badge.svg)

## Tech Stack
- Java 17
- Selenium 4
- TestNG
- Maven
- Allure Reports
- GitHub Actions CI

## How to Run
```bash
mvn test
```

## Framework Structure
- Page Object Model (POM)
- BasePage with reusable actions
- ConfigReader for externalized config
- Allure reporting with @Step annotations
